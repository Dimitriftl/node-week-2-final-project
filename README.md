## Getting Started

### Installing

#### Step 1: set the required environment variables

| ENV | Description | Required |
|---|---|---|
| `DB_USER` | User MongoDB | yes |
| `DB_PASSWD` |  Password for MongoDB | yes |

#### Step 2: installing

```sh
git clone https://github.com/Dimitriftl/node-week-2-final-project.git

npm install
npm start
```
